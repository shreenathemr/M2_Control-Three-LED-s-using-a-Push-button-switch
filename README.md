# M2_Control-Three-LED-s-using-a-Push-button-switch
## This repository contains my project for module 2
## Project must have below components
- Atmega 328 microcontroller
- LED
- Resistors (330 ohm) and (10 kilo ohm)
- Push buttons
- Fixed Volt
- Ground Resistor (0v)
